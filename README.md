# Прогнозирование уровня смертности с помощью Decision Tree и Random Forest

## Описание проекта
Цель работы — предсказать уровень смертности (`Death Rate Per 100,000`) на основе исторических данных IHME за 1970–2010 годы.

## Использованные данные
Набор данных: `IHME_GBD_2010_MORTALITY_AGE_SPECIFIC_BY_COUNTRY_1970_2010.csv`.
Признаки (Features):
* Год (Year)
* Пол (Sex)
* Возрастная группа (Age Group)
* Страна (Country)

## Что было сделано:
1. **Предобработка:** Очистка данных от пропусков (`dropna`).
2. **Кодирование:** Категориальные признаки переведены в числовые коды.
3. **Моделирование:** Обучены модели `DecisionTreeRegressor` и `RandomForestRegressor`.
4. **Валидация:** Данные разделены на обучающую и проверочную выборки (train/test split).

## Результаты
* **Decision Tree MAE:**3015.7477082908945
* **Random Forest MAE:**2398.289227269641

**Вывод:** Модель Random Forest показала более высокую точность (меньший MAE), так как она лучше справляется с переобучением, усредняя результаты множества деревьев.
